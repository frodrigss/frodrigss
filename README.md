### What I've been building

I lead the **full development lifecycle** of a unified corporate ecosystem for the custom planned furniture sector. Boa Vista Consulter is a centralized platform designed to replace fragmented processes across multiple operational units with a single, high-performance solution.

**From day one, the scope included:**

- 📋 **Requirements Gathering** — Corporate-level analysis of business needs across departments: operations, sales, production, and management. Translating those processes into technical specifications.
- 🏛️ **Solution Architecture** — Designed and implemented a scalable MVC architecture consolidating **CRM**, **executive dashboards (BI)**, **messaging**, and **scheduling/calendar** into a single unified system. Every architectural decision was reasoned and documented.
- 👨‍💻 **Development** — Full-stack implementation of backend APIs (ASP.NET Core), data layer (EF Core + SQL Server), and frontend (TypeScript + CSHTML + Bootstrap), with security protocols built in from the ground up.
- 🔒 **Security Implementation** — JWT authentication flows, BCrypt password hashing, and granular RBAC (Role-Based Access Control) to ensure data governance across different organizational roles.
- 📦 **Delivery & Presentation** — Formal presentation of deliverables to internal teams, aligned with business expectations and operational readiness.
- 🔧 **Continuous Updates & Maintenance** — Ongoing feature development, bug fixing, and performance improvements post-launch, keeping the platform aligned with evolving business needs.

**Measurable impact:**

> 📉 **80% reduction in operational failures** through process monitoring modules that surface bottlenecks in production, customer management, and internal communication in real time.

### ⚔️ SCAI — Imperial Access Control System

> _A production-grade REST API for logistics and access control management, built with enterprise patterns and a security-first approach._

SCAI demonstrates modern backend architecture applied to a domain-driven scenario. Authentication flows, layered design, automated testing, and interactive API documentation running in production.

**Architecture highlights:**

- 🏛️ **3-layer structure** — Presentation, Domain, and Data layers for clean separation of concerns
- 🔐 **Security** — JWT Bearer authentication + BCrypt password hashing + granular RBAC (roles: Admin, User)
- 🧪 **Test coverage** — 86% service-layer coverage with xUnit + Moq
- 📖 **Public Swagger** — Interactive docs with public `GET` endpoints and a login endpoint that unlocks `POST`, `PUT`, and `DELETE` for authenticated admins
- 🌱 **Data Seeding** — Automated database population for testing and staging environments
